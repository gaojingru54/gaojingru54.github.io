#ES6的两个新的变量声明关键字：const、let
const：表示常量，值在声明后不能修改。常量必须在声明时进行初始化，而且不能再次赋值。
例如： const i= 2; i = 3; // 报错，常量不能再次赋值
let：let用于声明块级作用域的变量，在块级作用域内部声明的变量只在该块级作用域内有效，而且不会被提升到函数作用域。

普通函数和箭头函数怎么转换
1.判断函数是否有参数。如果有参数，需要使用小括号包裹参数列表。如果只有一个参数，则可以省略小括号。 2.移除function关键字，并在参数和箭头之间添加一个箭头（=>）。 3.如果函数体只有一行代码，并且它是一个表达式，可以省略大括号和return关键字，并将该表达式放在箭头后面。 4.如果函数只有一个返回值，并且没有参数，可以使用隐式返回，即省略大括号和return关键字，并且将返回值放在箭头后面。

async 函数返回的是一个 Promise 对象。async 函数（包含函数语句、函数表达式、Lambda表达式）会返回一个 Promise 对象，如果在函数中 return 一个直接量，async 会把这个直接量通过 Promise.resolve() 封装成 Promise 对象。
await 在等 async 函数，但它等的实际是一个返回值。await 不仅仅用于等 Promise 对象，它可以等任意表达式的结果，所以，await 后面实际是可以接普通函数调用或者直接量。
例如：
function getSomething(){
return “something”;
}
async function testAsync(){
return Promise.resolve(‘hello async’);
}
async function test(){
let v1 = await getSomething();
let v2 = await testAsync();
console.log(v1,v2);
}
test();
console.log(‘我执行了’);
//执行结果为:
//我执行了
//something,hello async
