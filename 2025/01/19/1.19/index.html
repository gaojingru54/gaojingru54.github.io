##TypeScript
#Ts与js
1.TypeScript 属于静态类型的编程语言，JS 属于动态类型的编程语言。（静态：编译编译期做类型检查； 动态：执行期做类型检查。)
#ts使用
1.创建ts文件
2.将 TS 编译为 JS：在终端中输入命令，tsc **.ts
3.执行 JS 代码：在终端中输入命令，node **.js
#TS使用JS基本数据类型
1.number：let number: number = 23
2.string：let myName: string = ‘关注ve’
3.boolean：let isYouth: boolean = true
4.null:let a: null = null
5.undefined:let b: undefined = undefined
6.symbol:let s: symbol = Symbol()
##vue

2.组件化的架构设计,允许开发者创建可复用的UI组件

#ref和reactive
响应式的引用的原理:通过‍‍ proxy 对数据进行封装，当数据变化时‍‍触发模板等内容的更新。‍‍
响应式特性：这种引用当你去改变它的值的时候，‍‍页面是可以跟着它去做变化的。‍‍
ref用法：
import { ref } from ‘vue’;
const count = ref(0); // 创建一个Ref对象，初始值为0
console.log(count.value); // 访问Ref对象的值
count.value = 1; // 修改Ref对象的值
Reactive用于创建一个响应式对象：import { reactive } from ‘vue’;
const user = reactive({
name: ‘Alice’,
age: 30
});

},
deep: true,

}
watch: {
// 监听对象的某个属性，被监听的属性值发生变化就会执行函数，但获取的 newVal 值和 oldVal 值不一样
‘user.userNo’: {

console.log(newVal, oldVal);
},
deep: true,
immediate: true
}
}
